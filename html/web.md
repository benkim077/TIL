# Web

## 웹사이트의 구성 요소

### 웹사이트와 웹페이지

- 웹페이지

    - 글 그림 동영상 등 여러가지 정보

    - 다른 웹페이지로 이동하는 '링크'

- 웹사이트

    - 링크를 통해 여러 웹페이지를 연결한 것을 웹사이트라 함.

### 웹사이트의 구성 요소

- HTML (구조)

- CSS (표현)

- JavaScript (동작)

### 웹 사이트와 브라우저

- 웹사이트는 브라우저를 통해 동작

- 브라우저마다 동작이 약간 달라서 문제가 생기는 경우가 많다. (파편화)

- 해결책으로 웹 표준이 등장

## 웹 표준과 크로스 브라우징

- 웹에서 표준적으로 사용되는 기술이나 규칙

- 어떤 브라우저든 웹페이지가 동일하게 보이도록 함(크로스 브라우징)

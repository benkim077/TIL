The Document Object Model(DOM) 

### DOM은 Interface

- HTML 문서의 프로그래밍 interface.

- 프로그래밍 언어가 DOM 구조(Tree)에 접근할 수 있는 인터페이스를 제공

    - 문서 구조, 스타일, 내용 등을 변경할 수 있게 돕는다. 

### DOM은 웹페이지의 객체 지향 표현

- DOM은 웹 문서를 표현하고, 저장하고, 조작하는 방법을 제공한다.

- 자바스크립트로 DOM을 수정할 수 있다.

# Box Model

태그들의 부피감을 결정하는 것을 박스 모델이라 한다.

## Usage

### border, padding, margin 속성

각각 테두리, 안쪽 여백, 바깥쪽 여백을 지정할 수 있다. 

### width, height 속성

box의 가로 및 세로 길이를 설정할 수 있다.

<p>와 같은 block element에는 잘 적용되나,

<a>와 같은 inline element에는 적용되지 않는다.

# Float

글의 본문 안에 이미지를 삽입할 때 자연스럽게 삽입할 때 사용한다.

레이아웃을 잡을 때도 사용한다.

## Usage

```css
img{
    float:right;
}
```

이미지에 float를 주면 이미지 뒤에 위치하는 태그들은 피해가게 만든다.

```html
<p style="clear:both;">
```

피해가기 싫게 만들고 싶을 때엔 clear 속성을 사용한다.

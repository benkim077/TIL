# DB Database

> 모델 들어가기 전, DB 기초 키워드 학습

- DB : 체계화된 데이터의 모임

- DB 기본 구조 : 스키마와 테이블

## 스키마 Schema

- DB의 뼈대, 요약본

- 자료의 구조, 표현 방법, 관계 등을 정의

- **컬럼 정보와 데이터 타입**으로 구성

## 테이블 Table

- DB는 수 많은 테이블의 모임이다.

- 테이블 : 필드와 레코드를 사용해 조직된 데이터의 집합

### 필드

- 속성, 컬럼

- 각 **필드에 데이터 형식**이 지정된다.

### 레코드

- 튜플, 행(row)

- 테이블의 **실제 데이터가 레코드**에 저장

## Primary Key(PK)

- 기본 키

- 각 레코드의 고유한 값(식별자)

- 다른 항목과 중복될 수 없는 유일한 단일 값

## Query 쿼리

- 데이터를 조회하기 위한 명령어

- "쿼리를 날린다."(DB를 조작한다.)

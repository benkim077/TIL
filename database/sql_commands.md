# SQL

- DDL Data Definition Language

    - 데이터 정의 언어

    - 테이블, 스키마를 정의

- DML Data Manipulation Language

    - 데이터 조작 언어

    - 데이터를 조작(CRUD)

## SQL Syntax

- SQL 키워드(SELECT, INSERT, UPDATE)로 시작

- 세미콜론(;)으로 끝남

- 주석 `--`

- Statement(문) 

    - 독립적으로 실행할 수 있는 완전한 코드 조각

    - 문은 절로 구성

- Clause(절)
